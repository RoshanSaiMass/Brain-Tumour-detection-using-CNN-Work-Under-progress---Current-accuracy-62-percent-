## 📌 Project Overview

This project focuses on **Brain Tumor Classification from MRI images** using a **Convolutional Neural Network (CNN)**.

The objective is to build a deep learning model capable of classifying MRI scans into different tumor categories while experimenting with different preprocessing techniques, CNN architectures, and hyperparameters to improve generalization.

> **⚠️ Project Status:** Under Active Development

---

## 📂 Dataset

Dataset used:

**Brain Tumor Classification (MRI)**

The dataset consists of MRI brain scans categorized into different tumor classes and is used for supervised image classification.
